Triangulation Software

I am working on a software that allow the user to triangulate different kind of surfaces and to interact and make calculations about them. The project was previously written in Python and used Gtk for its user interface. I am currently working on a JavaScript version, which will make it easy for anyone to use the software just by going on my website.

Link to my GitHub page: https://github.com/do3244354/TriangulationSoftware

This is the poster for a short presentation I made at a math conference about the software:Triangulation_Software_Poster_DorotheeGrondin.pdf

Galois Theory Presentation

I was really surprised when I learned that there does not exist a formula to find the roots of any fifth degree polynomial. This encouraged me to study Galois Theory to really understand why such a formula is impossible. The presentation is currently only available in French.
